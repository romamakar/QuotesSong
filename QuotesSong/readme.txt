Квоти пісень v. 1.2

Дана програма створена для визначення квот пісень аналізуючи плейлисти в радіостанціях

Програма вимагає встановленого у користувача браузера Google Chrome 32-bit або Mozilla Firefox (32 або 64 біт) (в принципі, якщо ви не будете використовувати завантаження з Інтернету, то можете і не встановлювати)
Також перевірте версію .Net Framework. Мінімальна версія - 4.5

Вибірка мови програми реалізована тільки для української (якщо в назві композиції присутні літери і,ї,є)

-----Основні аспекти роботи з програмою------------------------------------------------------------------

На початку роботи програма запросить ввести початкові налаштування. Їх, при необхідності, можна змінити далі в програмі (Дані->Налаштування)

Головне вікно представляє собою вкладки з таблицею з декількома фільтрами (дата, параметр вибірки, порушники), що відображає квоти у процентному відношенні, у відповідності до списку мов(країн), вказаних у налаштуванні
та з графіком для зручного аналіізу

Пункт меню "Дані". Тут можна редагувати радіостанції та пісні

Пункт меню "Плейлист" - для редагування плейлисту, отримання його з Iнтернету (реалізовано поки що тільки для сайту radioscope.in.ua) та заповнення даних вручну

	- Заповнення даних вручну вимагає збереження наступних правил:
	1)Виберіть порядок назв аргументів у пісні (наприклад, назва - виконавець - тривалість - час)
	2)Далі заповніть текстове поле у такому форматі, який ви вказали в пункті 1, використовуючи розділювач "<пробіл>-<пробіл>" (наприклад, Тартак - Понад хмарами - 4:30 - 18:47) та нажміть "Enter" для додавання нової композиції.
	3)Також є можливість завантажити дані з текстового файлу (підтримується ТІЛЬКИ формат utf-8)

Пункт "Довідка" - інформація про програму

Версія 1.2
Додана підказка "Кількість виділених рядків"
Імплементація параметру "Квота по тривалості/кількості". Згідно закону враховуються пісні які мають тривалість 90 секунд і більше,
а також квоти визначаються по кількості пісень і їх співвідношенні до всієї кількості пісень в плейлисті.
Раніше була реалізована лише квота по тривалості пісень
Дублювання пісень при завантаженні з сайту/заповненні вручну ігноруються
Косметичне виправлення для вікна "Завантаження з сайту"
